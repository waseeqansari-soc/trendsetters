A) Installation - top

To install the "Excellez Design Express" template, follow these steps:

  1. Download the template package from ThemeForest and unzip it to your desired location on your computer.
  2. Upload the contents of the 'excellez-template' folder to your web server using an FTP client or your hosting provider's file manager.
  3. Ensure that the following files and folders are correctly uploaded:
    • index.html
    • css/styles.css
    • js/scripts.js
    • assets/
  4. Open your web browser and navigate to your website's URL to see the template in action.

B) Customization - top

Customizing the template is straightforward. Here are some common customization tasks:

1. Changing the Logo

To change the logo, replace the logo.png file located in the assets/images/ directory with your own logo file. Ensure the new logo file has the same name and format.

2. Updating Colors

To update the colors used in the template, open the styles.css file and find the color variables at the top of the file:

:root {
    --primary-color: #ff5722;
    --secondary-color: #333333;
    --background-color: #ffffff;
}

Change the values to your desired colors.

3. Editing Text

To edit the text content, open the index.html file and locate the sections you want to edit. For example, to change the main heading:

<h1 class="h1 text-color-white text-align-center-desktop text-align-center-mobile">Helping content creators transition to Web 3</h1>

Replace the existing text with your own content.

4. Updating the Favicon

To update the favicon, replace the favicon.png file located in the assets/ directory with your own favicon file. Ensure the new favicon file has the same name and format.

5. Adding New Sections

To add new sections to your page, copy an existing section in the index.html file and modify it as needed. For example, to add a new feature section:

<section class="new-feature-section">
    <div class="container">
        <h2>New Feature</h2>
        <p>Description of the new feature.</p>
    </div>
</section>

D) JavaScript - top

The project includes the following JavaScript files:

  • Swiper.js: For creating the slider component.
  • Bootstrap.js: For interactive components like modals and tooltips.
  • Custom scripts: Additional custom functionality.

1. Testimonials Slider

This section describes the functionality of the testimonials slider:

document.addEventListener('DOMContentLoaded', (event) => {
    let currentIndex = 0;
    const testimonials = document.querySelectorAll('.testimonial_content');
    const totalTestimonials = testimonials.length;
    const prevBtn = document.getElementById('prevBtn');
    const nextBtn = document.getElementById('nextBtn');

    function showTestimonial(index) {
        // Hide all testimonials
        testimonials.forEach((testimonial, i) => {
            testimonial.classList.remove('active');
            testimonial.setAttribute('aria-hidden', 'true'); // Accessibility improvement
        });
        // Show the selected testimonial
        testimonials[index].classList.add('active');
        testimonials[index].setAttribute('aria-hidden', 'false'); // Accessibility improvement
    }

    prevBtn.addEventListener('click', () => {
        currentIndex = (currentIndex === 0) ? totalTestimonials - 1 : currentIndex - 1;
        showTestimonial(currentIndex);
    });

    nextBtn.addEventListener('click', () => {
        currentIndex = (currentIndex === totalTestimonials - 1) ? 0 : currentIndex + 1;
        showTestimonial(currentIndex);
    });

    // Initialize the first testimonial
    showTestimonial(currentIndex);
});

3. Billing Toggle

This section describes the functionality for toggling between monthly and annual billing:

document.getElementById('bill-toggle').addEventListener('change', function () {
    if (this.checked) {
        document.querySelectorAll('.monthly-price').forEach(el => el.classList.remove('active'));
        document.querySelectorAll('.annual-price').forEach(el => el.classList.add('active'));
        document.getElementById('monthly-label').classList.remove('active-label');
        document.getElementById('annually-label').classList.add('active-label');
    } else {
        document.querySelectorAll('.annual-price').forEach(el => el.classList.remove('active'));
        document.querySelectorAll('.monthly-price').forEach(el => el.classList.add('active'));
        document.getElementById('annually-label').classList.remove('active-label');
        document.getElementById('monthly-label').classList.add('active-label');
    }
});

4. Logo Animation

This section describes the functionality for animating the partner logos:

window.addEventListener('load', function () {
    const logoList = document.querySelector('.logo_list');
    const logos = Array.from(logoList.children);

    // Duplicate the logos to ensure a seamless loop
    logos.forEach(logo => {
        const clone = logo.cloneNode(true);
        logoList.appendChild(clone);
    });

    let scrollPosition = 0;

    function animateLogos() {
        // Move the logos to the left by 1 pixel each frame
        scrollPosition += 1;

        // If the scroll position goes beyond the first set of logos, reset it to 0
        if (scrollPosition >= (logoList.scrollWidth / 2)) {
            scrollPosition = 0;
        }

        logoList.style.transform = `translateX(-${scrollPosition}px)`;

        // Call this function again on the next animation frame
        requestAnimationFrame(animateLogos);
    }

    // Start the animation
    animateLogos();
});
